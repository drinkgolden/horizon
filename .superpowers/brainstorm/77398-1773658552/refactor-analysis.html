4. CSS Naming — The Confusion

Three competing spacing systems coexist:
spacing-style — Liquid render helper → sets --padding-block-start/end
gap-style — Liquid render helper → sets dynamic --gap
layout-panel-style — Liquid render helper → sets flex direction + alignment + gap
Plus static scales:
--margin-xs / --padding-xs / --gap-xs — three families for the same thing
Plus logical properties:
--padding-block-start — from section settings, different system entirely
Worker token replacement: Collapse all three families into one --space-* scale. Use it for margin, padding, and gap alike.
