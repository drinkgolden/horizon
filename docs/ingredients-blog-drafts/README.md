# Ingredients Blog Drafts

These drafts are for Shopify Admin article publishing. The blog content itself does not live in the theme repository, so these files are the publish-ready source material.

## Current drafts

- `what-is-manuka-honey.md`
- `why-golden-uses-honey-instead-of-refined-sugar.md`

## Publish checklist

For each article:

1. Create a new article in the `ingredients` blog.
2. Use the `title`, `handle`, `seo_title`, `meta_description`, and `excerpt` from the file front matter.
3. Paste the article body into Shopify's article editor.
4. Add a relevant featured image.
5. Publish one article first, then the second a few days later rather than dropping both at once.
